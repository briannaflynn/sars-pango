######## Env setup

if not already, enter directory: cd sars-pango
Run: conda env create -f bioinformatics.yml
Activate: conda activate bioinformatics

Install via pip: 

pip install toytree
pip install plotly==5.1.0
pip install -U kaleido
pip install PyPDF2
pip install weasyprint
pip install jinja2

Do this after all installation for the env is completed:
- It is likely that the v1.14 of cairo will install instead of v1.16 on mac OS. You will have to go to https://anaconda.org/anaconda/cairo/files
and download the necessary files.
- brew install bedtools or just go get it
